# Dashy

Hey !
I'm tired to look at 5 different places to see my investments.

In close future, Dashy aims to centralise all of those and give a clear view on performances with detailed datas.

On the longer term, I plan to add a panel to run and manage trading bots and stategies.

## Running

For now, there is no official deployment method, and the app only runs in local.
A better solution will be added later but for now :
- install npm and cargo if they are not already
- go in `server` and execute `cargo run`
- go in `client` and execute `npm run dev`

And you're all set !

## Other works

This app is written with Sveltkit, and the UI is inspired by [@ichsanputr](https://github.com/ichsanputr)'s work [Takeka](https://github.com/ichsanputr/takeka). Even if I didn't straight up copied code it still was a heavy inspiration on the artistic direction of the app.
